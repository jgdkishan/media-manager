# media-manager
 # Setup Server
npm version: 8+
Node version: c14.17.4

 # Setup BE Server
 1. Configure the .env file in the root directory of the server repo.
 2. Run command "npm install"
 3. then run command "nodemon index" or "node index"
 4. listen to 8080


# Setup FE Server
 1. Configure the .env file in the root directory of the server repo.
 2. Run command "npm install"
 3. then run command "nodemon index" or "node index"
 4. Run `npm start`
 5. listen to 3000
